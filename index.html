Game Rules

  • The base of the map is a 5x5 (Easy) or 7x7 (Hard) square grid, depending on the selected difficulty.
  • The maps contain different numbers of:
    • Mountains
    • Bridges
    • Oases
    • Empty tiles
  • No specific requirements for placing railway elements exist, allowing for flexible gameplay.
  • Clicking on Cells: Clicking on a cell cycles through the different element types, with options to rotate or delete using the right mouse button.
  • Element Selection: After selecting a tile, elements can be chosen from a palette or switched between elements using the arrow keys.
  • Drag and Drop: Elements can be dragged and dropped onto a tile from a palette.
  • Drawing Railway Lines: The railway line is drawn by holding down the mouse button and moving across the cells (bonus points for this feature).
  • The game ends when the player correctly completes the puzzle according to the given rules.
    • Each cell must have two exit points, allowing the train to enter each neighboring cell from the correct direction.
    • Every cell that needs to be touched must be touched.
    • Every cell where no track can be placed remains empty.
